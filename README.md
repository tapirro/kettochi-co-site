# Kettochi

> Suplemento para el control glucémico con berberina, canela de Ceilán y cromo para apoyar niveles saludables de azúcar en sangre.

**[Visitar kettochi.co](https://kettochi.co)**

## Sobre este proyecto

Este repositorio contiene el sitio web de [Kettochi](https://kettochi.co), un recurso de salud basado en evidencia científica enfocado en **Control Glucémico**.

### Ingredientes clave

Berberina HCl, canela de Ceilán (*Cinnamomum verum*), cromo picolinato, gymnema sylvestre

## Tecnología

- HTML5 + CSS3 (sitio estático)
- Hospedado en GitHub Pages
- Sin dependencias de JavaScript frameworks
- Optimizado para Core Web Vitals

## Evidencia científica

Todo el contenido está respaldado por:
- Estudios clínicos publicados en PubMed
- Monografías de la EMA (European Medicines Agency)
- Guías de la OMS sobre plantas medicinales
- Base de datos TRAMIL de plantas del Caribe y Latinoamérica
